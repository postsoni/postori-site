# RCアーカイブ データベーステンプレート

## 📋 概要
このCSVファイルは、ラジコン機種情報を管理するためのデータベーステンプレートです。
Google Spreadsheet、Excel、LibreOffice Calcなどで開いて使用できます。

---

## 📁 ファイル
- `RC_Database_Template.csv` - データベーステンプレート（3機種のサンプル入り）

---

## 🔧 使い方

### **1. Google Spreadsheetで開く**
1. Google Driveにアクセス
2. 「新規」→「ファイルのアップロード」
3. `RC_Database_Template.csv`をアップロード
4. アップロードしたファイルを右クリック→「アプリで開く」→「Googleスプレッドシート」

### **2. Excelで開く**
1. Excelを起動
2. 「ファイル」→「開く」→`RC_Database_Template.csv`を選択
3. 文字コード: UTF-8を選択

### **3. LibreOffice Calcで開く**
1. LibreOffice Calcを起動
2. ファイルを開く
3. 文字エンコーディング: UTF-8を選択

## ✅ データ入力のポイント

### **必須項目**
- ID
- メーカー（日本語・英語）
- 機種名（日本語・英語）
- カテゴリー
- 発売年代
- 最終更新日

### **空欄でOKな項目**
- 型番（不明な場合）
- 生産終了年（現行販売中の場合）
- 全長・全幅・全高・重量（データ不明の場合）
- 公式URL・PDF（リンクがない場合）
- 修理実績（まだ修理経験がない場合）
- note・YouTube URL（コンテンツがない場合）

### **ID命名規則**
```
ラジコンカー: RC-CAR-001, RC-CAR-002, ...
ラジコンヘリ: RC-HELI-001, RC-HELI-002, ...
ラジコン飛行機: RC-PLANE-001, RC-PLANE-002, ...
```
